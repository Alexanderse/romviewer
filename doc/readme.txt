Source code is all .net 2010 and in the src folder.

The micromacro folder contains all the stuff that is required for the rombot to work with the romviewer. Copy this stuff out and add it to your micromacro folder. You may need to be careful with the userfunction stuff as it may be out of date.

Data contains a set of gamenodes and zones already and can be used as a starting point. Once you have the source code building, copy WorldData.dat to the folder where the executable is.
